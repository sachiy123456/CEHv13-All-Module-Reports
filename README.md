# CEH v13 Practical Lab Completion Report

## Executive Summary
This repository documents the structured completion of practical lab exercises aligned with the CEH v13 curriculum. The objective was to apply ethical hacking methodologies in a controlled environment to identify, exploit, and remediate security vulnerabilities across multiple domains.

The focus of this work was not only technical exploitation, but also understanding risk impact, attack surface exposure, and defensive countermeasures.

---

## Engagement Scope

The lab simulations covered the following security domains:

1. Footprinting and Reconnaissance  
2. Scanning Networks  
3. Enumeration  
4. Vulnerability Analysis  
5. System Hacking  
6. Malware Threats  
7. Sniffing  
8. Social Engineering  
9. Denial-of-Service  
10. Session Hijacking  
11. Evading IDS, Firewalls, and Honeypots  
12. Hacking Web Servers  
13. Hacking Web Applications  
14. SQL Injection  
15. Hacking Wireless Networks  
16. Mobile Platform Security  
17. IoT and OT Hacking  
18. Cloud Security  
19. Cryptography  

---

## Methodology Followed

Each lab was executed using a structured penetration testing lifecycle:

- Reconnaissance and Information Gathering  
- Threat Modeling and Attack Surface Identification  
- Service Enumeration  
- Exploitation of Identified Vulnerabilities  
- Privilege Escalation Techniques  
- Post-Exploitation Analysis  
- Risk Assessment and Mitigation Recommendations  

This approach ensured technical accuracy while maintaining alignment with industry-standard security assessment practices.
